Software Projects

Earth Bot

NOAA provides the raw imagery data from GOES-16 and GOES-17 satellites for free on AWS S3. It is relatively easy to combine the different wavelengths into a "color" image using Numpy and XArray

As always, there's always more to work on. On the list is GOES-17 support, and wrapping it all into a python package for ease of use for other people.

jonesAnalytics.ca

I love analyzing data and building dashboards. I have a ton of experience with Power BI, and I wanted to showcase the things I've built.

It is a Django web application, hosted through AWS Lambda. I'd say it's about 97% of the way there, I'm just working out the last minor issues. The dashboards are hosted through Power BI online, and displayed through iframes on the site.
